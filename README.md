# github_action_runner
Docker compose for Ubuntu github action runner self-hosted.

# .env
Create the .env file in same dir:

    ORG_NAME=<your-organization>
    ACCESS_TOKEN=<GITHUB_TOKEN>
    RUNNER_NAME=<runner-name>
